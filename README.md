# PraPdBL Frontend

## Description
Frontend web application for Project PraPdBL. Provides the user interface to interact with bookings, rooms, and users.

## Features
- Responsive UI for managing bookings.
- Room and user management screens.
- Client-side routing with React Router.
- Form handling and validation utilities.

## Tech Stack
- React 19
- TypeScript
- Vite
- MUI (Material UI)
- React Router DOM

## Installation
1. Install Node.js (LTS).
2. Install dependencies:
   - `npm install`

## Usage
- Start dev server:
  - `npm run dev`
- Build for production:
  - `npm run build`
- Preview production build:
  - `npm run preview`

## Environment Variables
Create a `.env` file in the project root if needed.

- `VITE_API_BASE_URL`: Base URL for the backend API (example: `http://localhost:5000`).
